# Jak na admin panel

## Přihlášení
- přihlášení probíhá přes `https://gttournament.cz/account`
- o oprávnění si napiš **@xvojta** na Discordu
- samotný admin panel je na `https://gttournament.cz/admin`

## Základní nastavení
- `Game page update`
- `Update a game`

## Typy eventů
- **Bracket**
- **Swiss**
- **Groups**
