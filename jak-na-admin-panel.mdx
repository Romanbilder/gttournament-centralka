# Jak na admin panel

<Steps>
  <Step title="Zadost o oprávnění" icon="key">
    O práva administrátora si musíš zažádat. Napiš **@xvojta** na Discordu.
  </Step>
  <Step title="Přihlášení do účtu" icon="user">
    Běžné přihlášení probíhá přes `https://gttournament.cz/account`.
  </Step>
  <Step title="Vstup do administrace" icon="screwdriver-wrench">
    Samotný admin panel najdeš na adrese `https://gttournament.cz/admin`.
  </Step>
</Steps>

## Správa turnaje a eventů

<CardGroup cols={2}>
  <Card title="Základní nastavení" icon="gear">
    Možnosti úprav v administraci:
    - `Game page update`
    - `Update a game`
  </Card>
  
  <Card title="Typy eventů" icon="sitemap">
    Systém podporuje tři základní formáty pavouků/skupin:
    - **Bracket** (Klasický pavouk)
    - **Swiss**
    - **Groups** (Skupiny)
  </Card>
</CardGroup>
