# GT Tournament Mintlify import

Tento balíček je připravený pro čistý Mintlify repozitář.

Obsahuje:
- `docs.json`
- kompletní MDX strukturu
- hotové stránky z přepsaných screenshot podkladů
- placeholder stránky tam, kde screenshoty neobsahovaly celý detail

## Jak to nasadit

1. Rozbal ZIP do rootu svého Mintlify repozitáře.
2. Pokud už v repu máš vlastní `docs.json`, udělej si zálohu a nahraď ho tímto.
3. Doplň logo, favicon a případné obrázky podle potřeby.
4. Spusť lokální preview přes svůj běžný Mintlify setup.

## Poznámka
Logo a images jsou záměrně ponechané prázdné, aby sis je doplnil v editoru podle potřeby.
